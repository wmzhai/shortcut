## webstorm

### 常用

* 新建文件：Command+'N'
* 查找打开文件：连按2次Shift



### emmet

输入div.row按tab键，自动生成

~~~
<div className="row"></div>
~~~

输入div.col-sm-6.col-sm-offset-3再按tab键，自动生成
~~~
<div className="col-sm-6 col-sm-offset-3"></div>
~~~
